# Sebenemaryam_Asnake_Lab1
Lab 1: Drone Pathfinding Using Search Algorithms

Student: Sebenemaryam Ashebir
Course: Intro to AI
Cohort: C

Overview

This project explores search algorithms for autonomous drone navigation in an environmental monitoring setting. The drone must move from a start to a goal position while avoiding obstacles such as forests, water, pits, and restricted zones.

Part A: Uninformed Search

Implemented BFS, DFS, DLS, and IDS using an object-oriented approach to study how different algorithms explore the environment without prior knowledge of the goal.

Part B: Informed Search

Extended the project with heuristic-based methods including Greedy Best-First Search, A*, UCS, and IDA*, comparing how heuristics improve efficiency and pathfinding performance.

Conclusion

This lab demonstrates how different search strategies perform under the same environment and highlights the efficiency gains achieved through heuristic-guided search.
