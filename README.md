# Sebenemaryam_Asnake_Lab1
Lab 1: Drone Pathfinding Using Search Algorithms

Student: Sebenemaryam Ashebir

Course: Intro to AI

Cohort: C


This repository contains two parts of search algorithms: uninformed and informed search (Part A and Part B). Part A implements BFS, DFS, DLS, and IDS, while Part B focuses on heuristic-based methods including Greedy Best-First Search, A*, UCS, and IDA*. This lab demonstrates how different search strategies perform in the same environment and highlights the efficiency gains achieved through heuristic-guided search.
